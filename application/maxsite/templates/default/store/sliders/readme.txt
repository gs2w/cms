Слайдеры tiny-slider
--------------------

В файле slider.php указываются параметры слайдера.

Слайдеры контента — tiny-slider-contentX выводят произвольный контент, который задаётся в файле content.php.

Соайдеры записей — tiny-slider-pagesX выводят записи. Формат задаётся в файле pages.php в виде юнита.

Чтобы сделать новый модуль слайдера, нужно спопировать наиболее подходящий вариант в modules. Можно в произвольный каталог, например modules/my/slider. В файле этого модуля index.php указать каталог
	
	[unit]
	module = my/slider
	...

Подключать как обычный модуль
	@module my/slider
